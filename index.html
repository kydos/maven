Kydos Maven Repository

To use this repository add to your build.sbt the following resolver:

resolvers += "kydos-repo" at "http://kydos.github.com/maven/snapshots"

As an example to add a dependency to escalier add the following to your build.sbt:

resolvers += "kydos-repo" at "http://kydos.github.com/maven/snapshots"
libraryDependencies += "net.icorsaro" % "escalier_2.9.2" % "0.6.0"
